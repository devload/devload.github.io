How It Works

SwiftCast acts as a local proxy between Claude Code and your AI providers.

1

Add Accounts

Add your Anthropic and/or GLM API accounts with their respective API keys.

2

Start Proxy

SwiftCast automatically configures Claude Code to route through the local proxy.

3

Use Claude Code

Work normally with Claude Code. Switch providers or models anytime from the SwiftCast UI.

4

Monitor Usage

Track your API usage in real-time. View detailed breakdowns by model, session, or day.
